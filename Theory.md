###	How Java is different from other programming languages?
Java is popular and most powerful programming language, created in 1995. It is used for mobile application(like android apps), desktop application, web application, web and application server  and database connection and much more.
Java is an object oriented programming language which gives a structured program and code reuseable.
Java is an platform independence (program can run on any os with help of java virtual machine).
It is easy to learn and simple to use (java syntax is simpler compared to other programming lang).
It is open source and free .
Multithreading :java allows multiple tasks runs at a same time its improve performance.
Standard librarie and API :java have built-in libraries and API for program needs
